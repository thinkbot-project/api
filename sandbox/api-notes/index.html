thinkbot API Reference v 1.0

thinkbot provides a friendly and powerful REST API that allows you to run sophisticated scientific software from just about anywhere.

Following convention, this API is organised as a set of resources you can manipulate. All requests use the application/json content type and go over https. The base URL to access the current version of the API is https://thinkbot.net/api/v1/.

If you have questions or feedback on the API, please get in touch with me.
